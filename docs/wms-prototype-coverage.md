# WMS 原型功能覆盖矩阵

| PRD | 原型模块 | 核心交互 |
|---|---|---|
| FR-001～002 | 主数据中心 | 仓库/库位树、商品包装条码、停用影响提示 |
| FR-003～006 | 入库管理 | ASN 扫描、收货差异、质检策略、生成上架任务 |
| FR-007～009 | 库存中心 | 多维查询、批次追溯、冻结与审计事务 |
| FR-010～017 | 订单、波次、补货、作业、复核发运 | 分配、释放、短拣、称重拦截、发运确认、取消拦截 |
| FR-018 | 盘点中心 | 盲盘、差异提交、复盘/审批 |
| FR-019 | 退货中心 | RMA 验证、序列号拦截、质检处置 |
| FR-020 | 任务中心 | 领取、转派、暂停和 SLA 监控 |
| FR-021 | 规则中心 | 冲突检测、样例试算、版本发布与回滚 |
| FR-022～023 | 驾驶舱、分析报表 | 多维指标、明细下钻、受控异步导出 |
| FR-024A | 用户管理 | 账号生命周期、认证来源、角色绑定、锁定/解锁、停用影响和工作交接 |
| FR-024B | 角色管理 | 角色复制、权限差异、影响用户、版本发布和历史版本追溯 |
| FR-024C | 权限管理 | 功能/数据/字段权限、职责分离、临时授权和有效权限模拟 |
| FR-025A | 审批中心 | 高风险申请、条件节点、SLA、通过/驳回和受控事务执行 |
| FR-025B | 审计日志 | 登录访问、安全配置、业务操作、数据访问、接口设备日志和受控导出 |
| FR-026 | 集成平台 | 接口健康、事件重试、请求追踪与对账 |
| FR-027 | 打印中心 | 模板、队列、失败重试、受控重打 |
| FR-028 | PDA 作业 | 扫描优先、离线缓存、恢复后幂等续作 |
